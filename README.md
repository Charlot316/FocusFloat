# FocusFloat

一个给 ADHD 工作流设计的 macOS 悬浮提醒应用。

它会直接读取 Apple 提醒事项，把“当前最该做的事”固定显示在桌面最上层；同时也可以从悬浮窗里新增提醒、选列表、设定时间，并和系统提醒事项双向同步。

## 现在已经有的能力

- 始终置顶的小窗，切到别的应用时也会继续显示
- 读取 Apple 提醒事项里的未完成任务
- 自动把最靠前的任务显示成“现在先做这件事”
- 监听提醒事项变化并自动刷新
- 直接在悬浮窗里新增提醒并设置时间
- 一键把任务标记为完成

## 运行方式

```bash
./scripts/run_app.sh
```

只想构建 `.app` 的话：

```bash
./scripts/build_app.sh
```

构建好的应用会放在：

```text
build/FocusFloat.app
```

## 首次启动

第一次打开时，macOS 会请求提醒事项权限。允许之后，应用就会开始同步你现有的 Apple 提醒事项。

如果你之前点过拒绝，可以在系统设置的“隐私与安全性 > 提醒事项”里把它重新打开。
